GitHub Pages 배포 실습 안내

현재 작성된 이 index.html 파일을 아래 단계를 통해 무료 웹사이트로 전 세계에 배포해 보세요!

  1. 1
    리포지토리 생성 GitHub에서 신규 Repository를 만듭니다. (이름 예: my-jeju-site)
  2. 2
    코드 업로드 (Push) 이 파일명을 꼭 index.html로 정한 뒤 리포지토리의 Root 폴더에 업로드합니다.
  3. 3
    Pages 기능 활성화 리포지토리 상단의 Settings > 좌측 Pages 메뉴로 이동합니다.
  4. 4
    배포 브랜치 선택 Build and deployment 섹션의 Source에서 Deploy from a branch를 선택 후, 아래 브랜치명을 main(또는 master)으로 설정하고 Save를 누릅니다.
  5. 5
    배포 완료 확인! 🎉 약 1~2분 대기 후 상단에 발급된 https://[유저ID].github.io/[리포지토리이름]/ 링크로 접속해 확인합니다.
